# antes
Web app about historical places.

### Estructura de Archivos
```
.
├── manual-pension-asistente
    ├── api                                 # ?
    ├── mobile                              # ?
    └── widgets                             # Contiene todos los widgets creados por Modyo CLI
        └── src
            ├── manual-pension-asistente    # Widget Manual de Pensión
            │   ├── components              # Componentes de VueJS
            │   │   ├── common              # Componentes aplicación
            │   │   └── pages               # Paginas principales del Widget
            │   └── store                   # Archivos del store separados (actions, mutations, state)
            ├── App.vue                     # Aplicación principal de VueJS
            ├── const.js                    # Contantes del proyecto
            ├── main.js                     # Punto de entrada Webpack   
            ├── README.MD
            └── shared                      # Librerías utilitarias compartidas
```
